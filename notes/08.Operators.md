# Operator Types

**Definition:**  
Operators are symbols used to perform operations on variables and values.

---

## Types of Operators

- Assignment ✔  
- Arithmetic  
- Comparison  
- Logical  
- Bitwise  
- Bit shift  
- instanceof  

---

## Assignment Operators

Used to assign values:

int a = 10;
a += 5;   // a = a + 5

---

## Arithmetic Operators

Used for mathematical operations:

+   // addition  
-   // subtraction  
*   // multiplication  
/   // division  
%   // modulus  

---

## Comparison Operators

Used to compare values:

==   // equal  
!=   // not equal  
>    // greater than  
<    // less than  
>=   // greater or equal  
<=   // less or equal  

---

## Logical Operators

Used for boolean logic:

&&   // AND  
||   // OR  
!    // NOT  

---

## Bitwise Operators

Operate on bits:

&   // AND  
|   // OR  
^   // XOR  
~   // NOT  

---

## Bit Shift Operators

<<   // left shift  
>>   // right shift  
>>>  // unsigned right shift  

---

## instanceof Operator

Checks object type:

if(obj instanceof Student) {
    // true if obj is Student
}



# Bitwise Operators

**Definition:**  
Bitwise operators work on **binary (bit-level)** representation of numbers.

---

## Bitwise AND (&)

- Returns 1 only if **both bits are 1**

Example:

x = 1 → 0001  
y = 3 → 0011  

x & y = 0001 → 1  

---

## Bitwise OR (|)

- Returns 1 if **either bit is 1**

Example:

x = 1 → 0001  
y = 3 → 0011  

x | y = 0011 → 3  

---

## Bitwise XOR (^)

- Returns 1 if **only one bit is 1 (not both)**

Example:

x = 1 → 0001  
y = 3 → 0011  

x ^ y = 0010 → 2  

---

## Bitwise NOT (~)

- Inverts all bits

Example:

x = 1 → 00000001  
~x   → 11111110 → -2  

---

## Left Shift (<<)

- Shifts bits to the left  
- Multiplies by 2

Example:

x = 2 → 0010  

x << 1 → 0100 → 4  

---

## Right Shift (>>)

- Shifts bits to the right  
- Divides by 2

Example:

x = 4 → 0100  

x >> 1 → 0010 → 2  

---

## Unsigned Right Shift (>>>)

- Shifts bits to right  
- Fills left side with 0

Example:

x = -4  

x >>> 1 → large positive number  

---

## Summary

| Operator | Meaning        |
|----------|--------------|
| &        | AND          |
| |        | OR           |
| ^        | XOR          |
| ~        | NOT          |
| <<       | Left Shift   |
| >>       | Right Shift  |
| >>>      | Unsigned Right Shift |



# Bit Shift Operators

---

## Left Shift Operator (<<)

- Shifts bits to the **left** by given number of positions  
- Inserts **0s on the right side**  
- Equivalent to **multiplication by powers of 2**

---

### Example

6 → 00000000 00000000 00000000 00000110  

6 << 1 → 00000000 00000000 00000000 00001100 → 12  

---

### Formula

6 << 1 → 6 * 2¹ → 12  
6 << 3 → 6 * 2³ → 48  

---

## Unsigned Right Shift Operator (>>>)

- Shifts bits to the **right**  
- Inserts **0s on the left side (higher-order bits)**  
- Always produces a **positive number**

---

### Example

12 → 00000000 00000000 00000000 00001100  

12 >>> 1 → 00000000 00000000 00000000 00000110 → 6  

---

## Key Points

- << → multiply by 2  
- >>> → divide by 2 (no sign preservation)  
- Inserts zeros
